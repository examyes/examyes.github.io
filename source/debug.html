调试方法¶

OpenGL¶

  • 官网的Common Mistakes
    • https://www.khronos.org/opengl/wiki/Common_Mistakes
  • 从 Qt 3D Render Pro Tips 获取的信息
    • apitrace （经测试可用）

    32位的程序使用x86的版本，64位的程序使用x64版本

    • vogl

    好像有几年不更新了，并且功能上貌似与apitrace比较接近

  • 从 apitrace 获取的信息
    • http://apitrace.github.io
  • 从 OpenGL官网 获取的信息
    • https://www.khronos.org/opengl/wiki/Debugging_Tools
  • 从以下网址获得的信息
    • OpenGL 图形库的使用（四十五）—— 实战之调试Debugging https://www.jianshu.com/p/7c81d561097e
  • RenderDoc
    • https://renderdoc.org/

    只支持OpenGL 3.2+ Core Profile，osgearth_viewer无法capture

  • nsight graphics
    • https://developer.nvidia.com/nsight-graphics

    对不严格遵循OpenGL规范(OpenGL 4.2 - 4.5, https://developer.nvidia.com/nsight-graphics-features)的程序支持很不好，比如osg…，我的天··· 还有一个Visual Studio(nsight VSE)版本的, https://developer.nvidia.com/nsight-visual-studio-edition

  • AMD codexl
    • https://gpuopen.com/compute-product/codexl/

    貌似只支持A卡
